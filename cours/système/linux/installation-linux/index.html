Outils Linux sous Macos

Macos est un système unix, mais ce n'est pas Linux. Son nom est Darwin. La plupart des commandes et des façons de faire Linux fonctionneront, il est donc courant de ne pas avoir besoin de Linux lorsque l'on est sous mac.

Cependant :

  • certaines options de commandes seront parfois différentes
  • les applications traitant du système ne seront pas identiques (le dossier /proc n'existe pas sous Macos par exemple)
  • les exécutables sont au format Mach-O et non elf
  • Les nouveau mac sont sous architecture ARM et non x86
  • le système est propriétaire, il est parfois difficile de
  • ...

Ces petites différences ne sont pas gênantes du moment qu'on ne crée pas d'application systèmes. Il existe de plus un excellent gestionnaire de paquet permettant d'installer quasi tous les outils Linux disponible : brew, que vous avez déjà du installé si vous avez suivi le tutoriel Ordinateur pour le développement
