## Structure du projet

Le nom du dossier du projet est `numérologie`{.fichier}. Sa structure finale est :

```text
.
├── back
│   └── numérologie.js
├── db.init.js
├── db.js
├── index.js
├── package.json
├── readme.md
└── static
    ├── index.html
    ├── main.css
    └── prénoms.html    
```

{% note %}
Ni le dossier `node_modules` ni les fichiers `package-lock.json`{.fichier} et `bd.sqlite`{.fichier} n'ont été représentés.
{% endnote %}
